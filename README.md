# 打卡系統
登入後讓使用者可以早晚打卡一次(共兩次)，若兩次打卡時間相隔低於8小時紀錄為缺勤，反之則記錄為全勤。

## 功能列表
* 使用者登入
* 使用者登出
* 使用者打卡
* 使用者修改密碼
* 點擊左上打卡系統，返回首頁


## 安裝
 1.打開你的 terminal，Clone 此專案至本機電腦
      
    git clone https://github.com/robert1074004/checkin
 2.開啟終端機(Terminal)，進入存放此專案的資料夾
 
    cd checkin
 3.安裝 nodemon
 
    npm install -g nodemon
 4.npm install所需要的套件
 
    npm install  
 5.啟動專案
 
    npm run dev
 6.複製顯示在終端機的網址，前往網頁
